# NKU_Parallel_2024

欢迎来到 **NKU_Parallel_2024** 代码仓库！这是我在南开大学并行程序设计课程（0921）中进行的项目和作业的集合。该仓库包含了我在学习并行编程过程中所编写的代码示例和实践项目。

## 项目简介

在并行程序设计课程中，我学习了如何利用多线程和分布式计算等技术提高程序性能。通过这个仓库，我希望能分享我的学习成果，并提供一些可复用的代码示例，以帮助其他同学和感兴趣的开发者。

## 技术栈

- **C++**：主要编程语言，用于编写并行程序。
- **OpenMP**：用于多线程编程的API，方便实现并行计算。
- **MPI**：用于分布式计算的标准，适用于多节点系统。
- **Python**：部分项目中用作数据处理和可视化。
